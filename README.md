# V2GTC Stack

Template for a website using Vue, Vite, Tailwind, Github Pages, and Catppuccin.

## Usage

1. Fork and Clone this repo
2. Run `npm install`
3. Run `npm run dev` to start the dev server
4. Modify the files to your liking
5. Edit the `vite.config.js` file and edit the `base` property to your repo name (if you want to use a custom domain, skip this and go to [Custom Domain](#custom-domain) then continue to step 6)
6. Push to your repo and enable Github Pages in the settings and select source as github actions
7. Enjoy your new website!

## Custom Domain

To use a custom domain, create a file called `CNAME` in the `public` folder and put your domain in it. Then, go to your domain registrar and create a CNAME record pointing to your Github Pages URL.
